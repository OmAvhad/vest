# Vest
Simplifying stock trading.

## Installation
```bash
git clone https://github.com/OmAvhad/vest.git
cd vest
```

### Backend
```bash
cd backend
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt

export FLASK_APP=app.py
export FLASK_ENV=development
export FLASK_DEBUG=1
flask run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
